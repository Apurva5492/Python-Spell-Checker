This project is a simple Spell Checker built using Python.
It takes a text input from the user, checks each word, and suggests the correct spelling if the word is misspelled.
The project uses the pyspellchecker library to identify and correct spelling mistakes.

This project is mainly created for learning purposes to understand:

Python classes and functions

Working with external libraries

Basic text processing
